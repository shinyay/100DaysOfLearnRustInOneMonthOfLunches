## Notes

**Result** - When an operation might succeed, but maybe not

### Result

`Result` is similar to `Option`

- `Option`
  - There might be something
    - `Some`: Value
    - `None`: No Value
- `Result`
  - It might be fail
    - `Ok`: Okay Result
    - `Err`: Error Result
