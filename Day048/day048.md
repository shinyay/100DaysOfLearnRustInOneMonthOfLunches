# Day 48

## Notes

Closures are like quick functions that **don't need a name**.
They use `||` instead of `()`. 

### Closures

Take a look at this, which the simplest sample for the Closure:

```rust
fn main() {
    let my_closure = || println!("Hello, Closure!");
    my_closure();
}
```
