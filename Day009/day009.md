### Declaring variables and code blocks

```rust
fn main() {
    let this_year = 2023;
    println!("Hello, Rust in {}", this_year);
}
```

Since Rust 2021, you can write it like the following:

```rust
fn main() {
    let this_year = 2023;
    println!("Hello, Rust in {this_year}");
}
```
